You can clone the sbt-native-packager GitHub repo and modify the rst files in the src/sphinx/ directory.

The site is generated with the sbt-site plugin. You’ll need the following dependencies installed:

  • python-sphinx
  • python-pip

and the sphinx_bootstrap_theme library available via pip:

pip install sphinx_bootstrap_theme

See the sbt-native-packager developer guide for more information.
