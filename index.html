This is the documentation for the sbt-native-packager plugin. This plugin aims to allow easy generation of native packages for all major operating systems, including:

  • Windows MSI files
  • Debian DEB packages
  • RedHat RPM packages
  • Docker images
  • Universal (Zipped) packages for easily-launchable start scripts
